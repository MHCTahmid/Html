Table Attributes

< table border=? > (only takes integer values (For examle: 10)
Sets the width of the border around table cells

< table cellspacing=? >
Sets amount of space between table cells

< table cellpadding=? >
Sets amount of space between a cell's border and its contents

< table width=? > (In text use percentage For examle: 60% , In image case pixels For examle: 60)
Sets width of the table in pixels or as a percentage. Sample

< tr align=? > (For examle: center, left or right)
Sets alignment for cells within the row (left/center/right)

< td align=? >
Sets alignment for cells (For examle: left/center/right)

< tr valign=? >
Sets vertical alignment for cells within the row (top/middle/bottom)

< td valign=? >
Sets vertical alignment for cell (top/middle/bottom)

< td rowspan=? >
Sets number of rows a cell should span (default=1)

< td colspan=? >
Sets number of columns a cell should span

< td nowrap >
Prevents lines within a cell from being broken to fit

< table bgcolor=? >
Sets background color of a table

< tr bgcolor=? >
Sets background color of a table row

< td bgcolor=? >
Sets background color of a table row data Sample

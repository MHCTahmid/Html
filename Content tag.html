Block or Box Element

The div tag defines a division or a section in an HTML document. The div tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The div tag is easily styled by using the class or id attribute . Any sort of content can be put inside the div tag! Used to format block content with CSS.
< div > < / div >

< div class='city' > < / div >
Classes are not unique. You can use the same class on multiple elements.You can use multiple classes on the same element.

< div id='dhaka' > < / div >
IDs are unique.Each element can have only one ID.Each page can have only one element with that ID.

Anchor Tag(Links)

The anchor tag defines a hyperlink, which is used to link from one page to another.

Attributes

href

< a href="http://www.google.com/" > < /a >
< a href="contact.html" > < /a >
< a href="mailto:email@example.com" > < /a >
< a href="tel://+88017102223334" > < /a >

download

The download attribute specifies that the target (the file specified in the href attribute) will be downloaded when a user clicks on the hyperlink. The optional value of the download attribute will be the new name of the file after it is downloaded. There are no restrictions on allowed values, and the browser will automatically detect the correct file extension and add it to the file (.img, .pdf, .txt, .html, etc.). If the value is omitted, the original filename is used.

hreflang

The hreflang attribute specifies the language of the linked document. This attribute is only used if the href attribute is set.

media

rel

The rel attribute specifies the relationship between the current document and the linked document. Only used if the href attribute is present.Search engines can use this attribute to get more information about a link!

HTML Boilerplate

In computer programming, boilerplate code or boilerplate refers to sections of code that have to be included in many places with little or no alteration. It is often used when referring to languages that are considered verbose, i.e. the programmer must write a lot of code to do minimal jobs.In web development if we type ! and press Tab key we will find the boilerplate of HTML.

Example : 

    < !DOCTYPE html > --- Decleartion of HTML-5 

     < html lang="en" > --- Specifies the language

        < head >

            < meta charset= "UTF-8" > character sets --- Specifies the character encoding for the HTML

            < meta http-equiv= "content type, default style, refresh", content="text gives the value associated with the http-equiv attribute" > --- provides an HTML header for the inforamtion value of the content attribute 

            or

            < meta http-equiv="refresh" content= "5" --- every 5 second the websit will auto refresh

            < meta name= "author, keywords, applicable name generator, description," content="text gives the value associated with the name attribute" > --- specifies a name for the metadata

            < title > Title of the website < / title >

        < / head >

        < body >

        The main content of the HTML document or the section of the HTML document that will be directly visible on your web page. This tag is also commonly referred to as the body element.

        < / body >

        < / html >
